GENERAL USAGE NOTES
===================

The main Program is called Favourite_Movies.py.

In the sourcecode are defined some Movies, which will be shown by image in
a browser window. Double-Clicking the image will show a trailer of the movie.

The code uses two modules:
    1. media.py, which includes the movie class
    2. fresh_tomatoes.py, which prepares the movie-data für html output, which will be
        shown in the browser.
